A) Page Templates - top

Duplex comes with five custom page templates; Homepage, Blog Post, Archive, Contact and Full Width. These templates can be used on individual pages to alter the display or functionality.

Homepage Template

The homepage template is of magazine style in nature and displays an array of content in various formats.

Blog Post Template

The blog post template displays the content of each page/post and includes sections for related posts and comments.

Archives Template

The archive template is intended to be used as either an date archive, category archive, author archive, tag archive or for any other situation where posts would need to be grouped e.g. search results.

Full Width Template

The Full Width template allows the creation of a page without a sidebar - the main content will span the full width of the site.

Contact Form Template

Duplex comes packed with a fully functioning contact form. The form includes jQuery validation to enhance user experience and the recipient email can be configured from the within the code. At the top of this template you will see two references to an email address. You will need to alter these to a working email address in order to receive emails.

B) CSS Files & structure - top

The Duplex Template includes a number of CSS files that are used at various times depending on the site's display. All CSS files, with the exception of the main CSS file (style.css) can be found in the Template's /css/ folder.

  • Reset.css - The goal of a reset stylesheet is to reduce browser inconsistencies in things like default line heights, margins and font sizes of headings, and so on. More information can be found here.
  • Style.css - The template's main stylesheet.
  • 960.css -The 960 Grid System is an effort to streamline web development workflow by providing commonly used dimensions, based on a width of 960 pixels. More information can be found here.
  • ie.css - CSS styles specific to Internet Explorer
  • superfish.css - Essential CSS styles for the dropdown menus
  • text.css - Default typography settings

CSS Structure

Each page template is separated into three parts, #header, #the_body and #footer.

#header

The #header represents everything from the top of the website to the main navigation. This element stays the same throughout each page template.

#the_body

The #the_body represents everything in the centre of the website. Anything in-between the main navigation and the footer

#footer

The #footer represents everything at the bottom of template. This remains the same for all page templates.

Click the image below to see a full-sized overview of the CSS classes and ID's.

The slider

Each slide is in an element with class ".item". Append a ".item" section after the previous panel to create more slides.

The navigation for the slider is in "#slider_nav". This can only contain 6 items. So you should only have 6 panels. you should only need to change the image source here.

Featured Post Carousel

The carousel is made up of content columns using the class ".column". Add more containers with the class ".column" to add more content to the carousel.

The sidebar tabs

Each tab section contains normal lists with data. Change this data to your own by replacing what is already there. The tab navigation is located just above the first tab code. You can change the tab text here.

C) JavaScript - top

This Template imports various Javascript files.

  1. jQuery
  2. Superfish
  3. JQuery Easing Plugin
  4. Template's Custom JS
  5. jQuery UI
  6. JQuery Validation Plugin
  7. jQuery Cycle Plugin
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Superfish controls the display of the dropdown menus. More info here
  3. The JQuery Easing plugin allows developers to add effects to animations
  4. The Template's custom JS file defines various opacity transitions and dropdown menu settings.
  5. The JQuery UI library allows developers to add advanced effects with jQuery
  6. JQuery Validation Plugin handles the validation of the inbuilt contact form
  7. The jQuery Cycle Plugin is a slideshow plugin that supports many different types of transition effects.
